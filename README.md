# FundamenTUNE
- - -
Tired of boring, old equal temperament tuning?  
Suffer no more, start using **FundamenTUNE** today for all your crazy tuning needs
- - -

## Future commands:
- ` > copy <note> <new note>`
- ` > del <note>`
- ` > name <note> <new name>`
- ` > base <note A> <note B>`
- ` > find <interval name> <fundamental> <depth>`
- ` > tune <note A> <note B> <delta interval>`
